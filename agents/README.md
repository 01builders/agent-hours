# Agent Submissions

This folder contains agents built during the workshop.

## How to Submit

1. Create a folder with your name: `agents/your-name/`
2. Copy `TEMPLATE.md` into your folder and rename it to `README.md`
3. Fill out every section in the template
4. Add your agent code to the same folder
5. Push your changes

### Example structure

```
agents/
  TEMPLATE.md
  README.md
  jane-doe/
    README.md      <-- filled-out template
    agent.py       <-- your agent code
    ...
  john-smith/
    README.md
    src/
      ...
```

## Guidelines

- One folder per person (or team).
- The filled-out template is required. It helps everyone understand what you built and why.
- Keep your agent self-contained within your folder where possible.
